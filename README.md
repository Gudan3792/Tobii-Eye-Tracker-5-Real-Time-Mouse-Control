# Tobii-Eye-Tracker-5-Real-Time-Mouse-Control
Control your mouse cursor in real-time using Tobii Eye Tracker 5 gaze data.

## Requirements

- Python 3.10.11 (32-bit)
- Tobii Eye Tracker 5
